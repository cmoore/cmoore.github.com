--- layout: default title: Royale With Cheese ---
Royale with Cheese is a locally hosted web service. This basically means that you run it on your local computer like any other application, but you interact with it through a web browser.

Features

  • Services » Send statuses to Twitter,identi.ca, and Facebook from one interface.
  • Timeline » All statuses are listed in chronological order in a single timeline.
